# 管理开发环境

通常有一些 Python 包只在你的开发环境中需要，而不是在生产环境中，例如单元测试包。 Pipenv 使用–dev 标志区分两个环境。也就是说一个–dev 参数，帮你在同一个虚拟环境中又区分出了开发和非开发环境。
