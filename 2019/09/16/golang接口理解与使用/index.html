interface是一种类型

首先 interface 是一种类型，从它的定义可以看出来用了 type 关键字，更准确的说 interface 是一种具有一组方法的类型，这些方法定义了 interface 的行为

go 允许不带任何方法的 interface ，这种类型的 interface 叫 empty interface
